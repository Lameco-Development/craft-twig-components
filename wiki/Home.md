# Requirements

- Craft CMS >= 4.9
- PHP >= 8.3
- AlpineJS >= 3
    - Collapse

# Tailwind

This plugin uses Tailwind. To include the default classes in the compilation of TailwindCSS add the templates folder as a source in your (S)CSS file using:
```scss
@source "../../../vendor/lameco/craft-twig-components/**/*.twig";
```
